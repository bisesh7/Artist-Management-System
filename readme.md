# Artist Management System

Task given by Cloco Inc.

## Stack

Frontend:

- React
- React-Bootstrap
- Axios
- Formik & Yup

Backend:

- Node.js
- Express.js
- SQLite
- JWT Authentication
- Multer

## Installation

Clone the repository:

```
git clone https://github.com/bisesh7/Artist-Management-System
cd Artist-Management-System
```

Install dependencies:

Backend:

```
cd backend
npm install
```

Frontend:

```
cd frontend
npm install
```

## Run the app

Backend:

```
cd backend
npm run dev
```

Frontend:

```
cd frontend
npm start
```
